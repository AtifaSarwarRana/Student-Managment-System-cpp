# Student Management System

A console-based Student Management System developed in C++ using Object-Oriented Programming principles and file handling.

## Features

- Add Student
- View All Students
- Search Student by ID
- Update Student Information
- Delete Student Records
- Input Validation
- File Handling for Permanent Storage

## OOP Concepts Used

- Encapsulation
- Inheritance
- Abstraction
- Classes and Objects

## Classes Implemented

1. Person
2. Student
3. Validation
4. StudentManager
5. Menu

## File Storage

Student records are stored in:

students.txt

Format:

ID,Name,Age,Department,CGPA

Example:

101,Ali,20,CS,3.5

## Technologies Used

- C++
- Object-Oriented Programming
- File Handling
